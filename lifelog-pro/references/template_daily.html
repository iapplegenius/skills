Docker 用户权限 & Volume 挂载
第 3 次遇到同一问题（4/20, 5/3, 今天）。根本原因：不理解 Dockerfile 中 USER 指令的执行时机和 volume mount 的权限继承机制，每次靠搜索临时解决但没有真正理解。
立即解决：
RUN mkdir -p /var/lib/data && chown -R appuser:appuser /var/lib/data
USER appuser # 必须在 COPY/RUN 之后，CMD 之前
