核心能力

把每一天变成
可分析的数据

LifeLog Pro 不需要你手动记录任何东西。它自动读取你今天在电脑上做的一切，然后帮你看见自己。

AUTO 🤖
全自动数据采集
运行一个 Python 脚本，自动读取 Shell history、浏览器历史、AI agent 对话记录、软件使用情况、今日编辑文件。不需要粘贴，不需要手动输入。
INSIGHT ⚡
能量曲线可视化
从命令时间戳提取二阶信号：每小时工作密度、专注块数量、高峰/低谷时段、分心指数。用一条 SVG 曲线看清你今天的真实状态。
MEMORY 📖
历史关联 + 拐点分析
自动扫描过去 30 天的日报，识别项目停滞、技能短板复现、专注度拐点。不只是"你最近在努力"，而是"你的专注度在周三崩了，那天发生了什么"。
LIVE 🌐
实时新闻 × 专家委员会
每次生成日报，自动抓取今日科技、财经、科学、健康新闻，交给五位领域专家结合你的数据给出动态建议——不是固定模板，是今天的世界 + 今天的你。
ACTION 🚀
明日行动闭环
综合专家给出的不是鼓励，是一个格式化的行动指令："明天 9 点，做 X，预计 30 分钟，目标：可验证的结果 Y。" 日记从回顾变成驱动。
GROWTH 📅
日报 / 周报 / 月报
说"生成周报"，自动聚合过去 7 天日报，输出 7 天能量曲线叠加图、项目时间矩阵、技能短板周追踪，以及周度版专家委员会建议。
运行原理

六个阶段
从数据到洞见

每次触发 LifeLog Pro，自动按序执行以下流程。

PHASE 00
判断报告模式
根据触发词自动判断：日报（今天）、周报（本周）、月报（本月），进入对应的处理流程。
PHASE 01
自动采集数据
运行 collect.py，从本机读取 Shell history、浏览器历史、agent 对话、软件使用。失败的数据源自动跳过。
PHASE 02
解析 + 二阶信号
分类命令、提取项目名、识别错误重试，同时计算命令时间间隔，构建能量曲线和专注块数据。
PHASE 03
历史关联 + 拐点
扫描过去 30 天日报，运行拐点识别算法，找出专注度变化节点、项目停滞警告、技能短板复现。
PHASE 04
实时新闻抓取
Web Search 抓取今日科技/财经/科学/健康新闻，提取标题、摘要、来源，作为专家委员会的信息基础。
PHASE 05
专家委员会建议
五位专家各自综合用户数据 + 新闻，输出具体建议。综合专家压轴，给出明日第一行动。
PHASE 06
生成 MD + HTML
输出 daily_YYYY-MM-DD.md 和 .html 双文件。HTML 包含六个 Tab，含能量曲线 SVG 和专家卡片。

使用方法

三步开始
零配置

LifeLog Pro 不需要注册账号，不需要 API Key，不需要配置文件。直接对 Claude 说话即可。

STEP 01
安装 skill（一次性）
将 lifelog-pro.skill 文件上传到你的 Claude skill 目录，或由管理员安装。安装后无需其他配置。
STEP 02
对 Claude 说任意触发词
在 Claude Code、claude.ai 或任何集成了 Claude 的环境中，直接说"生成今天的日记"或"lifelog-pro"，skill 自动激活。
STEP 03
等待生成，下载文件
Claude 自动采集数据、抓取新闻、生成日报，最终输出 .md + .html 两个文件供下载。整个过程约 1-3 分钟。
可选
手动补充内容
若自动采集遗漏了某些内容，可以额外粘贴 history 文本或上传截图，Claude 会自动合并处理。
可选
指定日志存储路径
将每天生成的 daily_*.md 文件存放在固定目录（如 ~/lifelog/），skill 会自动扫描并用于历史关联分析。目录越完整，分析越精准。
